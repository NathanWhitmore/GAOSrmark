# GAOSrmark
Automated photo-resight analysis with RMARK

### How to download
First install the "remotes" package via:
```
install.packages("remotes")
```

in Program R then run:
```
library("remotes") 
install_github("NathanWhitmore/GAOSrmark")
```

and then load the package via:
```
library("GAOSrmark")
